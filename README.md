# Monetary policy models

This project presents some of the most relevant DSGE (Dynamic Stochastic General Equilibrium) models used in monetary policy and macroeconomic analysis. Each folder contains the implementation of the model in both Matlab/Dynare and Python. The Python versions use the Sequence-Space Jacobian library, developed by Auclert and co-authors for HANK (Heterogeneous Agent New Keynesian) models.

For each model, the corresponding research paper is provided, explaining its equations, constraints, preferences, economic environment, and other theoretical foundations. The goal is to give users a solid theoretical reference while facilitating the implementation and understanding of each model.

The idea of this project is to start with the most canonical models in the macroeconomic literature and gradually add new features to better understand the mechanisms underlying the models used by central banks around the world. These models are highly robust and accurate, but the number of assumptions, equations, and economic mechanisms they incorporate makes them quite complex.
